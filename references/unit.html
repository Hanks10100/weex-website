CSS number 单位

仅仅一个数字。用于 opacity，lines等。

有时值必须是整数，例如：lines。

CSS percentage 单位 (暂不支持)

表示百分比值，如“50％”，“66.7％”等。

它是 CSS 标准的一部分，但 Weex 暂不支持。
